# City Events Guide (Arabic)

Multi-page site built with HTML/CSS/JS + Bootstrap, ready for GitHub Pages.

## Structure
- index.html — Home (featured slider, quick categories, latest events, participants)
- events.html — All events with filters (category/date)
- event.html — Event details (+ add to calendar, share, related events, map image)
- about.html — About the guide (team, policy)
- contact.html — Contact form (validation + Bootstrap alerts)
- /css/styles.css — styles
- /js/main.js — JS logic & dataset
- /img — images folder (optional)

## Run locally
Open `index.html` in your browser.

## Deploy to GitHub Pages
1) Create a public repo and upload all files (root must contain index.html).
2) In repository **Settings → Pages**: Source = `main`, Folder = `/root`, Save.
3) Wait ~1 minute; your site will appear at: `https://USERNAME.github.io/REPO-NAME/`
